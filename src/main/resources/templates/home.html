Efficient Contact Management Awaits You

Discover a streamlined way to manage your contacts with our cutting-edge Smart Contact Manager. Our platform offers seamless contact storage, advanced search functionality, and robust security features. Join us and elevate your contact management experience to the next level!

SCM

The Smart Contact Manager efficiently organizes user contacts, offering features like secure login, pagination, search by name or email, Excel export, dark mode, and feedback via email. It’s built with Spring Boot and Tailwind CSS.

Features

  • Secure User Login
  • Contact Pagination
  • Contact Export to Excel
  • Dark Mode Support

Additional Features

  • Custom Login via GitHub and Google
  • Spring Security for URL Protection
  • Email Verification
  • Password Recovery

Tech Stack

  • Spring Boot
  • HTML/CSS
  • JavaScript
  • Thymeleaf
  • Tailwind CSS
